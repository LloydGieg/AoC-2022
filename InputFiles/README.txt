This directory should contain the following files:

* answerkey.txt:
    A comma-delimited text file containing each day's correct output. Each day should have its own line in the
    following format:

        d[x], [part 1 answer], [part 2 answer]

        where:

        x = The day (i.e. 'd1' for day 1).
        part 1/2 answer: Just paste the answer. No need to quotes, data types, etc.

* A text input file for each day called 'd[x].txt' containing the input for the day
